# site-bootstrap

Command terminal sass : ```sass --watch assets/sass/index.sass:assets/css/style.css```

# Pre-requis
  - Node JS V 14


# Install

```
git clone https://github.com/PurpBrain/site-bootstrap.git
cd site-bootstrap
npm i
```

# Run

run:
```
npm start
```

developpement:
```
npm run dev
```

run sass (compile)
run:
```
npm run sass
```
